## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

- Node.js v20.10.0
- Vite v5.0.8
- Vue v3.3.11

### 取得專案

```
git clone git@github.com:ellaYang1227/typescript-group-work.git
```

### 移動到專案內

```
cd typescript-group-work
```

### 安裝套件

```
npm install
```

### 運行專案

```
npm run dev
```

### 運行 eslint

```
npm run lint
```

### 運行 prettier

```
npm run prettier
```

### 瀏覽器開啟專案

```
http://localhost:5173/
```
